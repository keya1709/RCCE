# 🌱 Reforestation Carbon Credit Estimator (RCCE)

> A web-based tool to estimate carbon sequestration and carbon credits generated through reforestation projects.

---

## 📌 Project Overview

The **Reforestation Carbon Credit Estimator** is a beginner-friendly web application that allows users to simulate how much carbon dioxide (CO₂) can be absorbed by planting trees over time.

Users can:
- Select different tree species 🌳  
- Assign weight distribution (%)  
- Enter land area and project duration  
- View carbon sequestration results visually  

---

## 🚀 Features

- 🌿 Multiple species selection  
- ⚖️ Custom weight distribution for species  
- 📊 Interactive charts (annual & cumulative carbon)  
- 📈 Real-time validation and feedback  
- 🎨 Dark/Light mode toggle  
- 📁 Export results as CSV  
- 🔄 Reset and re-run simulations  

---

## 🧠 How It Works

1. User inputs:
   - Area (in hectares)
   - Duration (years)
   - Tree species + weights

2. System:
   - Uses biomass growth data from JSON
   - Applies carbon fraction
   - Calculates yearly sequestration

3. Output:
   - Total carbon captured
   - Year-wise data
   - Graphical visualization
